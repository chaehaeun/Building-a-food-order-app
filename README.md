# Building a Food Order App

유데미 '리액트 완벽 가이드' 강의를 보며 만든 프로젝트입니다.

## 🚩 규칙

1. 강의 넘버마다 이슈 생성 후 강의 한 개 넘어갈 때마다 커밋
2. 메모하고 싶은 것이 있으면 강의 넘버 이슈에 메모
3. 메모한 이슈와 목차의 체크리스트 링크 연결
4. 오래 기억해두고 싶은 내용일시 TIL 레포에 항목 추가 후 내용 기입
